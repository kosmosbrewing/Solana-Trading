## 1. 결론 요약 (3줄)

1. **엔지니어링 저력은 증명됐다** — 측정·승격·손실통제 기계가 실전에서 작동했고, 그 덕에 자본을 보존한 채 가설을 $0 에 기각할 수 있었다.
2. **Edge 저력은 미증명이며, 증거는 누적적으로 부정적이다** — 사전 등록 가설 7전 7패, Phase 0 통과 후보 0, 구조 벽 3개가 반복 확인됨.
3. **폐기 확정 전 남은 $0 게이트는 정확히 1개 (H-007a)** — 그 결과가 "재개할 좁은 길 1개" 와 "edge 탐색 종료" 를 가른다. 결정은 의견이 아니라 그 숫자가 내리는 것이 이 프로젝트의 규칙이다.

## 5. 3대 구조 벽 — "더 노력하면" 으로 안 풀리는 것들

1. **속도 벽**: break-even latency 음수 — 0초 체결로도 KOL-follow 는 불가 (`EDGE_AUDIT_REPORT` E3-E6). $58/월 예산으로 latency 군비경쟁 불참은 Mission v2 로 공식화됨.
2. **고정비 벽**: 왕복 실행비 0.0027 SOL = 0.02 ticket 의 13.6% — 모든 미세 edge 를 지배. ticket 상향은 gate 통과 cohort 전제 (아직 0개).
3. **관측 벽**: KOL 매수 pool 의 65.8% 가 pump.fun bonding curve (WS parser 미지원, coverage 상한 ~34%) + 구독 지연으로 covered anchor 33% 가 pre-entry 창 손실 + **신호 불일치** — 싸게 볼 수 있는 것(가격 경로)은 기각됐고, edge 후보(holder/dev 시계열)는 유료 수집 + 미구축 파서 필요 (`coverage-postfix-2026-06-13` FINDINGS §5). Helius free 1M = 현 구성 observe run 월 ~10일 한계.

## 6. 폐기해도 소멸하지 않는 것 (보존 자산)

- **작동이 증명된 손실 통제 기계** — Real Asset Guard + survivor universe 필터. Mission v2 목표 ① 은 이미 달성물이다.
- **음의 지식 (영구)**: KOL-follow / consensus / rebound / 가격·거래량 모멘텀 / majors 추세 — 이 예산·이 시장에서 전부 불가함을 실측으로 확정. 재제안 금지 원장: `HYPOTHESES.md`.
- **전이 가능한 인프라**: 실시간 WS 수집, 온체인 tx 실행 + 비용 분해, 원장-대-온체인 정산 정합성 (reconciliation), 크래시 세이프 저널링, TimescaleDB 시계열, 크레딧 귀속 — 타 도메인 (예: 결제/정산 시스템) 재사용 가치.
- **자본**: wallet floor 무위반 + 예비금 $1,000 무손실.

## 7. 결정 트리 — 마지막 $0 게이트: H-007a

> **H-007a** (`HYPOTHESES.md` 등록, 2026-06-13): 이미 수집된 `token-quality-observations` (5,970 rows) 의
> `operatorDevStatus` / `riskFlags` 를 markout forward outcome 에 join —
> **"dev/quality 차원에 ex-ante 예측력이 있는가?"** 비용 $0 / API 0 / 로컬 데이터만.

```
H-007a 실행
├─ 신호 없음 (flag 별 forward 분포 무차별, CI 0 포함)
│   → H-007 기각 = 저비용으로 열 수 있는 마지막 문이 닫힘
│   → **edge 탐색 폐기 확정** (데이터가 내린 결정)
│   → 처리: 봇 = 종료 or "생존 base layer" 로만 보존 / 인프라·음의 지식 문서화
│   → Helius 결제·레버 2·$1,000 전부 불집행
└─ 신호 있음 (특정 flag cohort 의 forward 분포가 유의하게 양수 방향)
    → holder 시계열 수집 투자 (Helius Developer $49/월) 의 최초 정당화
    → 좁은 1개 경로로 재개: H-007 본검정 사전 등록 → 기존 promotion gate 불변
    → 단 $1,000·ticket 상향은 여전히 mirror gate 이후에만
```

**권고**: 지금 상태로 "재개" 는 데이터가 지지하지 않고, "폐기" 는 H-007a 를 돌리기 전엔 지적으로 미완이다. → **H-007a 를 먼저 실행하고, 그 결과로 결정한다.** 이것이 이 프로젝트가 지켜온 규칙 (돈·감정이 아니라 사전 등록된 검정이 결정) 의 마지막 적용이다.
